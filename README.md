# Retail Sales SQL Analysis Project

🔹 Overview
This project analyzes retail sales data using SQL to generate business insights.

🔹 Tools Used
- MySQL Workbench
- SQL
  
🔹 Key Analysis Performed
- Total Sales Calculation
- Monthly Sales Trends
- Product Ranking using RANK()
- Running Total using Window Functions
- Region-wise Sales Analysis

🔹 SQL Concepts Used
- Aggregate Functions (SUM, AVG)
- Window Functions (RANK, OVER)

🔹 Project File
- retail_analysis.sql → Contains all SQL queries
